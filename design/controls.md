## User Manual: System Controls

This section outlines how to operate the Automotive Air Conditioning Control Panel using the keypad interface.

System Modes
Key 3 (Mode Select): Switches the system between Automatic and Manual operation.

Auto: The fan starts and stops automatically based on the temperature sensor.

Manual: The user controls the fan power directly using the keypad.

Temperature Settings
Key 6 (Temp Up): Increases the target setpoint temperature by 1°C.

Key 4 (Temp Down): Decreases the target setpoint temperature by 1°C.

Fan Control (Manual Mode)
Key 1 (Fan Power): Toggles the fan ON or OFF. (Note: This function only works when the system is in Manual Mode).

Fan Speed & Configuration
Key 2 (Speed Up): Increases the fan speed by 10% (Maximum 100%).

Key 8 (Speed Down): Decreases the fan speed by 10% (Minimum 10%).

Key 5 (Direction/Mode Toggle): Switches the operation mode between Heating and Cooling.

Cooling Mode: Fan spins in the forward direction.

Heater Mode: Fan spins in the reverse direction.
